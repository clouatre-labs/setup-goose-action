# Security Policy

## Reporting Vulnerabilities

**Email:** hugues@linux.com (or use GitHub Security Advisory)

**Scope:** This action only (not Goose CLI)

**Response:** 48 hours acknowledgment, 7-14 days fix

## Supported Versions

| Version | Supported |
|---------|-----------|
| v1.x    | ✅        |
| < v1.0  | ❌        |

## Out of Scope

- Goose CLI bugs → https://github.com/block/goose/security
- Workflow security → See [examples/](examples/)
